== Allgemeines
* Open-Source
* gRPC (Remote Procedure Call)
* Schneller als REST
* Requests und Responses sind vordefiniert

== Alternativen

* REST
* SignalR
* JSON-RPC

== Vorteile
* Hohe Performance (Kleinere Messages als in REST)
* Built-in Code Generation (Cross-platform)
* Mehrere Verbindungsmöglichkeiten
* HTTP/2
